# 🔐 Tcpreplay Cyber Security Mini Project

## 📌 Description
This project demonstrates how to use Tcpreplay in Kali Linux to simulate network traffic and analyze packet replay attacks.

## 🛠 Tools Used
- Kali Linux
- Tcpreplay
- Wireshark

## ⚙️ Steps Performed
1. Captured network packets using Wireshark
2. Saved packets in .pcap file
3. Replayed packets using tcpreplay
4. Analyzed traffic behavior
